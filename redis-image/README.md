# redis

Example for building redis in docker

## Getting started

- cd `redis-image`
- Build image `docker build .`
- Run image `docker run <container id>`

### Useful schortcuts

- Use `ctrl`+`d` to exit container
- Use `ctrl`+`c` to stop container
